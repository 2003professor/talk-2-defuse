How to Play

Two Roles, One Mission

The Executor sees the bomb — wires, buttons, keypads — but has NO manual.

The Instructor has the bomb defusal manual but CANNOT see the bomb.

You must communicate through the chat to defuse the bomb before time runs out!

Step by Step

  1. Describe — The Executor describes what they see (bomb shape, size, colors, modules).
  2. Identify — The Instructor looks up the bomb type in the Index tab.
  3. Instruct — The Instructor reads the manual and gives step-by-step defusal instructions.
  4. Execute — The Executor follows the instructions carefully. Click components to interact.

Tips

  • Hover over bomb components to see labels — use these to describe clearly.
  • A confirmation popup appears before each action — take a moment to verify.
  • Wrong actions cause strikes. Too many strikes = explosion!
  • Watch the timer — it changes color as time runs low.
